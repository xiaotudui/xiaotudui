# 数据集标记工具

前面，我们讲解了常见的两种目标检测数据集：VOC 和 COCO。

这一部分，我们来讲解 目标检测数据集的常见标注工具。

通过这些标注工具，我们可以看到数据的标注文件是什么样的，以及他们在图片上的表示形式。这样更加直观。

因为数据集标注，涉及到大量的数据，所以一般标注工具都是在本地完成的。

所以，我们可能先需要进行一些本地环境的安装。

首先，我们需要安装下 Anaconda。https://www.anaconda.com/download/success


## LabelImg

LabelImg 支持导出VOC，YOLO数据格式。

## Label Studio

Label Studio 是一个开源的标注工具，支持多种任务的标注，包括目标检测，图像分割，文本分类等。

```shell
pip install label-studio
```


![label-studio](@site/static/object-detection-basic/object-detection/label-studio-install.png)

安装完成后，输入`label-studio`，回车后，会自动打开浏览器，可以看到如下界面。
我们点击右下角的 `Sign up`，在本地进行账号的注册，这个账号是存储在本地，大家不用担心。

![label-studio](@site/static/object-detection-basic/object-detection/label-studio-login.png)
