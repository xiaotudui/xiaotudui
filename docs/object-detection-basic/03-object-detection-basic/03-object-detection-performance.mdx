# 目标检测性能指标

前面，我们训练完了模型。

现在，我们来学习一些性能指标，通过这些性能指标，我们可以评估目标检测模型性能的好坏。
在了解评估之后，我们在后面的部分，对模型进行优化，这样就可以定量分析优化效果。

## IoU

如何衡量模型的预测框和真实框的相似度？

这是我们首先要思考的问题。


IoU，即Intersection over Union，交并比。

交并比，是用来衡量两个目标框的相似度。

mAP，即Mean Average Precision，平均精度均值。
