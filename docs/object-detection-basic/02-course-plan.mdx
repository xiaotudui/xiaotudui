# 课程安排

思来想去，这个教程还是作为 目标检测 的入门教程。

在入门教程中，主要就是对 目标检测这个领域 有个大概但是清晰的认识。

以下，是这个课程的规划思路。

1. 理解是什么目标检测，到底是做什么的。只有理解了目标检测到底是做什么的，才能知道这个领域知识是否适用于你的课题/工作，或者是否是你感兴趣的领域。

:::tip[土堆碎念]
我一直说，深度学习主要就分为三个大模块：数据集，模型，损失函数。对于不同的领域，我们就是首先要了解这三个大模块。
:::
2. 了解目标检测数据集。通过了解目标检测数据集，我们可以进一步加深对目标检测做什么的有个更清晰的了解。
其次，通过目标检测数据集，我们可以清晰的认识到 模型 的输入和输出到底是什么样的。

3. 在讲解完数据集之后，我们进一步来带大家看下标注工具，带大家标注自己的数据集。（我们如果需要解决自己的课题/工作，肯定是要用到自己的数据集）

:::tip[土堆碎念]
其实标注数据这一部分，可以放到后面来说。但我思来想去，放在前面说的好处，是让大家动手去标注，加深对数据集格式的理解。动手标注的话，肯定比单纯看视频讲解格式要好得多。
:::
